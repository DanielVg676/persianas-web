# Docs del proyecto

Este directorio registra decisiones importantes, estructura y notas de implementacion para la landing.

## Indice

- `architecture.md`: organizacion de carpetas y reglas de ubicacion.
- `design-system.md`: paleta, tipografia, espaciado y direccion visual.
- `components.md`: componentes reutilizables, variantes y estados esperados.
- `content-and-data.md`: tono, placeholders y datos comerciales centralizados.
- `assets.md`: logos reales, imagenes stock provisionales y reglas de reemplazo.
- `accessibility-performance.md`: reglas de accesibilidad, SEO local y rendimiento.
- `implementation-log.md`: bitacora de cambios relevantes.
